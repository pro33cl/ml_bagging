# ml_bagging
Notebook que implementa modelos Bagging para predecir renuncias de clientes. Incluye análisis exploratorio, Decision Trees, SVM, SMOTE para balancear clases, BaggingClassifier, RandomForest, GridSearchCV, evaluación con métricas (ROC, AUC, F1) y selección de los clientes con mayor probabilidad de renuncia.
